Electronics

There are 3 main electronic bodies reponsible for 3 separate electronic functions of the Electronic Safe.

The first one, or the Long Breadboard serves as the Input Board for the Code of the Safe. When the Huzzah board, or the disattachable electronic board on the breadboard receives a signal from an online database (see more on the Coding section below), it sends a signal to the series of buttons, that if pressed, send signal from the Breadboard to the Metroboard.

The second system is the Accelerometer Breadboard. This Accelerometer device that is mounted on the Breadboard sends information of the rotational position of the board to the Metro board that it shares connection with the Long Breadboard.

The third is the Anti-Theft System of the Safe. The breadboard of the system is attached to its own metroboard. It consists of a Buzzer, a capacitance sensor and an LED Strip. The system is designed to start ringing its buzzer and make its LED Strip light up if it ever senses an electric charge caused by the human finger (touching the metal part of the sensor). Essentially, if anyone tries to manually open the Safe, an visual and sound alarm will go off. This Process is asisted by Coding. Aluminum foil was attached to the Safe's door, as well as a wire going from the circuit to the Door. This wire will sense the capacitance charge in the Aluminum Foil.

Arduino Codes

The first two systems are interconnected, but use separate code. The first system, uses a simple Firebase Code that instructs the Huzzah board to turn on when the online database (Firebase Database) instructs it to (accomplished by in person input).

The Second Code commands two motors, when both the correct sequence of buttons are pressed as well as the Accelerometer board is tilted at the right angle, to start executing their specific functions, either to work together to open and close the Safe. The code also commands two LED lights to turn on

The third and final code commands the buzzer and LED strip connected to its own independent Breadboard and Metro Board, to set off, when it senses a capactitative charge time of more than 2000 (caused by touching the Capacitance Sensor).

Assembling and Using my Project:

Two N20 Motors were used for this project as part of the machinery. They were both connected to washers that were then attached to the 3D Printed Wheels.

The 3D printed Door was wrapped in aluminum foil to act as a Capacitance Sensor. The other part of the Sensor being the brown metal sheet.
